# Bundled Assets

Place your production assets here:

- Character images: `../composeResources/drawable/<image_key>.png`
- Audio clips per voice pack: `audio/<voice_pack_id>/<clip_id>.mp3`
- Optional narration backtrack (looped during playback): `audio/backtrack/the_uncertain_quest.mp3`
- Optional custom fonts: `fonts/`

Current placeholders keep the scaffold pathing stable.
